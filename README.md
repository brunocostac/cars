# cars


## Especificações de desenvolvimento

* Arquitetura utilizada: VIP
* Desenvolvimento de telas com ViewCode
* Biblioteca de terceiros: SdWebImage, SkeletonView e ProgressHud
* Banco dados local com CoreData
